# Malingstatistikk V8.2

Grafisk finpuss av mini-dashboardene for andre varehus.

- Leverandørlogoene er redusert og tilpasset kortbredden.
- Tekst, KPI-brikker og søyler er komprimert uten overlapp.
- Alle fire varehuskort er dimensjonert for A3 liggende på én side.
- Print-layouten beholder tre leverandører per varehus og to varehus per rad.
- Mobilvisningen har egne mindre logo- og tekststørrelser.
