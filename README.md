# Malingstatistikk V6

Oppgradert møte- og utskriftsversjon.

## Endringer
- Oppstart viser «Vennligst vent – henter siste rapport og produktdata» i stedet for Admin-knapp.
- A3-rapporten er komprimert til én side.
- «Dagens budskap» er fjernet.
- Nederste del av utskriften er delt i tre kategorier:
  - Maling / dekkbeis / beis
  - Vindu / dør
  - Murmaling
- Hver kategori viser topp 3 produkter med produktbilde.
- Produktkort viser grafisk:
  - antall spann
  - fortjeneste i kroner
  - fortjenesteprosent
  - fargekodet fortjenestemåler
- Regionrangeringen er komprimert til en smal stripe nederst.
- Eksisterende Neon-, Blob-, produktregister- og automatiske bildeoppslag er beholdt.

## Deploy
Last opp innholdet i denne mappen til roten av GitHub-repositoryet. Vercel bygger Next.js-prosjektet automatisk.
