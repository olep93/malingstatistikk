# Malingstatistikk V11 – Mobile First

V11 er en full mobil-revamp bygget på V10.12.1.

- Gjenoppretter alt sideinnhold på mobil.
- Fast, trygg bunnnavigasjon med Dashboard, Produktsalg og Historikk.
- Vareområde velges i en kompakt mobilvelger øverst.
- Kontrollpanel er tilgjengelig i toppfeltet.
- Dashboard, KPI-er, leverandørkort og produktkort er optimalisert for små skjermer.
- Produktmaster vises som redigerbare kort uten horisontal scrolling.
- Historikk, rapportarkiv, importjobber og brukeradministrasjon er responsive.
- Desktop- og A3-visning er beholdt.
- Den gjenopptakbare historikkanalysen fra V10.12.1 er beholdt.
