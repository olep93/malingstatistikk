# Malingstatistikk V8 – Executive Dashboard

V8 bygger videre på V7.5 og gir en samlet grafisk oppgradering.

## Oppgradert
- Leverandørkort bruker tydelig leverandøridentitet: Infra lilla, Butinox blå og Jotun gul.
- Mini-dashboard for andre varehus viser både omsetningsandel og fortjenesteandel per leverandør.
- Mini-dashboard viser omsetning, fortjeneste og fortjenesteprosent kompakt.
- Bedre kontrast, skygger, typografi og produktkort.
- Mobilvisningen stabler mini-dashboardene riktig.
- A3-utskriften beholder fargene og viser de grafiske søylene på én side.

## Deploy
Last opp innholdet i prosjektmappen til roten av GitHub-repositoryet. Vercel bygger prosjektet automatisk.
