# Malingstatistikk V12.1 Enterprise

Samlet versjon med V12-ytelseslag, utskriftsforbedringer, leverandørmiks, produktmaster-rettelser og utvidet automatisk audit.

Audit kjører nå seks faser: oppslag, navneoppdatering, automatisk klassifisering, aliaser, kontrollmerking og opprydding. Oppslag kjøres i små batcher for å unngå Vercel-timeout.
